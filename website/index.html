Translations tool

  1. Be patient! On some browsers this application takes quite some time to load.
  2. Play around and report bugs to our bugtracker
  3. Initially you see only the keys to be translated. It is not a bad idea to klick the "en" button as the english translation is the most complete one normally.
  4. If the key contains %0, %1, ... those symbols must be in the translation, too as they are wildcards for content that is added at runtime.
  5. This tool does not require a login. Still it is protected against vandalism to a degree as you can store states of modification to our versioning control system by clicking "store changes".
  6. If you want to be mentioned in the List Of Contributors please provide both your name and a mail adress in the "store changes"-dialogue.
